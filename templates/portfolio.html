About This Project

This web application demonstrates encryption/decryption and password hashing/verification using Flask, bcrypt, and the cryptography library (Fernet).

Features

  • Encrypt and decrypt text using Fernet (symmetric encryption from the cryptography library)
  • Hash passwords securely with bcrypt
  • Verify passwords against bcrypt hashes
  • Modern, colorful, and interactive frontend

Encryption & Decryption

Encryption: Enter any text and click Encrypt to see the Fernet-encrypted ciphertext.
Decryption: Paste a Fernet-encrypted ciphertext and click Decrypt to reveal the original plaintext.
The app uses Fernet from the cryptography library, which provides secure, authenticated symmetric encryption.

Password Hashing

Password Hashing: Enter a password and click Hash Password to generate a bcrypt hash.
Password Verification: Enter a password and a bcrypt hash to check if they match.
This ensures passwords are never stored or transmitted in plain text.

Technologies Used

  • Python (Flask)
  • bcrypt
  • cryptography (Fernet)
  • HTML, CSS, JavaScript

How to Run

  1. Clone the repository or download the code.
  2. Install dependencies: pip install flask bcrypt cryptography
  3. Set environment variable FERNET_KEY.
  4. Run python app.py and open http://127.0.0.1:5000
