About This Project

This web application demonstrates two separate security tasks: Cryptography (Encryption/Decryption) and Password Hashing & Verification using Flask, bcrypt, and the cryptography library (Fernet).

Task 1: Encryption & Decryption (Cryptography)

Cryptography is the science of securing information by transforming it into an unreadable format, only reversible by authorized parties. In this project, symmetric encryption is used, meaning the same key is used for both encryption and decryption.

Encryption and decryption is done using AES (Advanced Encryption Standard) through the Fernet implementation.

Encryption

Encryption is the process of converting readable data (plaintext) into an unreadable format (ciphertext) using a secret key. Only someone with the correct key can decrypt and access the original information.

Decryption

Decryption is the reverse process of encryption. It transforms the encrypted data (ciphertext) back into its original readable form (plaintext) using the same secret key.

Features

  • Encrypt text using Fernet (symmetric encryption from the cryptography library)
  • Decrypt Fernet-encrypted ciphertext to reveal the original plaintext
  • Secure, authenticated symmetric encryption

How It Works

Encryption: Enter any text and click Encrypt to see the Fernet-encrypted ciphertext.
Decryption: Paste a Fernet-encrypted ciphertext and click Decrypt to reveal the original plaintext.
The app uses Fernet from the cryptography library, which provides secure, authenticated symmetric encryption.

Technologies Used

  • Python (Flask)
  • cryptography (Fernet)
  • HTML, CSS, JavaScript

Task 2: Password Hashing & Verification

Password Hashing is a security technique where a password is transformed into a fixed-length string using a one-way function. This ensures that even if the data is compromised, the original password cannot be easily retrieved. Hashing is essential for securely storing user credentials.

Features

  • Hash passwords securely with bcrypt
  • Verify passwords against bcrypt hashes
  • Ensures passwords are never stored or transmitted in plain text

How It Works

Password Hashing: Enter a password and click Hash Password to generate a bcrypt hash.
Password Verification: Enter a password and a bcrypt hash to check if they match.
This ensures passwords are never stored or transmitted in plain text.

Technologies Used

  • Python (Flask)
  • bcrypt
  • HTML, CSS, JavaScript

How to Run

  1. Clone the repository or download the code.
  2. Install dependencies: pip install flask bcrypt cryptography
  3. Set environment variable FERNET_KEY.
  4. Run python app.py and open http://127.0.0.1:5000
